# Programação - Técnico em Informática (SENAC)

## Objetivo

Este repositório reúne os projetos, exercícios e atividades desenvolvidos durante as Unidades Curriculares **UC9 a UC16** do curso **Técnico em Informática do SENAC**. Ao longo dessas unidades foram aplicados conceitos de algoritmos, desenvolvimento de software, banco de dados, desenvolvimento web, testes e implantação de sistemas, culminando na elaboração de um projeto integrador.

---

## Conteúdo

Durante as unidades curriculares foram desenvolvidas atividades relacionadas aos seguintes temas:

* Desenvolvimento de Algoritmos
* Programação para aplicações desktop
* Desenvolvimento de aplicações web
* Banco de Dados Relacionais
* Modelagem de Dados e SQL
* Testes e Implantação de Sistemas
* Documentação de Software
* Desenvolvimento de Sites (HTML, CSS e JavaScript)
* Manipulação de elementos gráficos para web
* Projeto Integrador

---

## Tecnologias Utilizadas

* C
* Python
* SQL
* MySQL
* HTML
* CSS
* JavaScript
* Git e GitHub

## Unidades Curriculares

* **UC9:** Desenvolver Algoritmos
* **UC10:** Desenvolver Banco de Dados
* **UC11:** Executar Teste e Implantação de Aplicativos Computacionais
* **UC12:** Codificação, Manutenção e Documentação de Aplicativos para Desktop
* **UC13:** Codificação, Manutenção e Documentação de Aplicativos para Internet
* **UC14:** Manipulação e Otimização de Elementos Visuais para Web
* **UC15:** Desenvolvimento e Organização de Sites
* **UC16:** Projeto Integrador Assistente de Desenvolvimento de Aplicativos Computacionais

---

## Objetivo do Repositório

Este repositório tem como finalidade documentar a evolução dos estudos e reunir os projetos desenvolvidos durante as UC9 a UC16, demonstrando a aplicação prática dos conhecimentos adquiridos no curso Técnico em Informática do SENAC.
